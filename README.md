# LC3-TicTacToe
Written in assembly, a terminal, text-based game of tic-tac-toe (2 player). 

<img width="736" height="953" alt="image" src="https://github.com/user-attachments/assets/d9e7cbd1-6b02-438b-a901-4ccaf433f93d" />

## Features

The program utilizes a variety of subroutines, handling:
- Board and rules display
- Input validation
- Condition checking
    - for wins
    - for ties
## Simulator Requirements
This program was built using [LC3 Tools (v2.0.2)](https://github.com/chiragsakhuja/lc3tools)
### Special requirements

Ensure window size is big enough to view the whole tic-tac-toe board. Default terminal, window size, and zoom should suffice.
- otherwise, options to zoom in or out under the "view" tab in the upper left-hand corner
